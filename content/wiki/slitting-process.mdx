# Coil Slitting Process: Step-by-Step Technical Guide

The coil slitting process converts a wide master coil into multiple narrower slit strips in a single continuous pass. Understanding the process parameters and their interdependencies is essential for achieving consistent strip quality, minimizing knife wear, and running an efficient operation.

---

## Process Overview

A slitting line operates as a continuous flow process with five functional zones:

1. **Uncoiling** — master coil is loaded and unwound at controlled tension
2. **Straightening** — coil set is removed before the slitter head
3. **Slitting** — circular knife pairs shear the material into multiple strips
4. **Scrap handling** — edge trim is separated and removed
5. **Recoiling** — individual slit strips are rewound under controlled tension

Each zone must be correctly set up and matched to the others. A tension mismatch between zones, for example, will cause strip camber, edge waviness, or recoil shape defects even when the slitter head setup is correct.

---

## Step 1: Master Coil Preparation

Before loading, verify:
- Coil OD and weight are within machine capacity
- Material grade, thickness, and hardness match the work order
- Coil is free from telescoping, edge damage, or surface defects that would cause feed problems
- ID is correct for the uncoiler mandrel

**Coil set measurement:** Determine the coil set direction (inner wrap tends to curve inward; outer wrap curves outward) before threading. The straightener must be set to compensate for the specific coil set magnitude — a coil from a tight OD will have more set than one from a large OD.

---

## Step 2: Knife Arbor Setup

The arbor setup determines the width of each slit strip and the cutting quality. Errors at this stage propagate to every strip on every coil until the arbor is re-set.

### Width Calculation

For each strip position, calculate the spacer width to position each knife pair:

```
Spacer width = Strip width − (upper knife width + lower knife width) / 2
```

For precision work, account for knife clearance (which adds to the nominal strip width) and for material springback.

### Horizontal Clearance Setting

Set the horizontal clearance between upper and lower knife faces for the specific material:

| Material Type | Clearance (% of thickness) |
|--------------|---------------------------|
| Mild carbon steel | 8–12% |
| Stainless steel 304/316 | 10–15% |
| Aluminum alloy | 5–8% |
| Copper | 4–6% |
| Electrical steel | 6–10% |
| Pre-painted / galvanized | 10–14% |

**Example:** 1.5 mm stainless steel at 12% clearance = 0.18 mm horizontal gap per knife pair.

### Knife Penetration (Overlap)

Set the vertical overlap between upper and lower knife cutting edges:
- Thin material (< 1 mm): 0.1–0.3 mm overlap
- Medium gauge (1–4 mm): 0.3–1.0 mm overlap
- Heavy gauge (> 4 mm): 1.0–2.5 mm overlap

Insufficient overlap leaves material uncut or creates a tear rather than a clean shear. Excessive overlap increases cutting force, accelerates wear, and can cause knife deflection.

### Arbor Torque and Clamping

After loading the knife and spacer stack, clamp the arbor with the specified torque. Under-torqued arbors allow knife migration under cutting loads, shifting strip widths during production. Verify the full stack is clamped uniformly — uneven clamping creates differential knife face contact and promotes galling.

---

## Step 3: Line Threading

Threading sequence follows the material path from uncoiler to recoiler:

1. Thread the strip leader through the straightener with rolls in open position
2. Close straightener rolls to setting calculated for material thickness and coil set
3. Advance strip through the loop pit — establish a stable loop before threading the slitter head
4. Thread the strip leader through the knife and spacer stack, maintaining knife engagement
5. Thread edge trim over the scrap separation guides to the scrap winder or chopper
6. Thread slit strips individually through separators to the recoiler
7. Secure strip ends to recoiler mandrels with correct wrap direction

**Do not jog the line to thread at production speed.** Threading at low speed (5–15 m/min) gives the operator control to correct strip path errors before they propagate to the knife stack.

---

## Step 4: Setup Verification at Low Speed

Run the first 5–10 meters at slow speed (10–20 m/min) before accelerating to production speed. Check:

- All strips are tracking correctly without edge contact on guides
- Scrap is feeding cleanly to the scrap winder
- No strips are wrapping onto the arbors
- Initial slit edge quality — measure burr height on the first 2 meters
- Strip widths match work order within tolerance

If burr height exceeds specification, stop and verify clearance setting before continuing. Burr formation at low speed is amplified at production speed as temperature and cutting forces change.

---

## Step 5: Production Run

### Speed Ramping

Accelerate to production speed in controlled ramp — typically 10–30 seconds to full speed. Rapid acceleration creates tension spikes in the loop that can cause strip flutter or edge damage at the slitter head.

### Tension Control

Uncoiler brake tension and recoiler winding tension must be matched to material properties:

| Material Type | Uncoiler Tension | Recoiler Tension |
|--------------|-----------------|-----------------|
| Mild steel 1–3 mm | 15–30 N/mm width | 20–40 N/mm width |
| Stainless steel | 25–50 N/mm width | 30–60 N/mm width |
| Aluminum alloy | 8–15 N/mm width | 10–20 N/mm width |
| Copper | 10–20 N/mm width | 15–25 N/mm width |

Excessive recoiler tension elongates the strip, causing width narrowing and potential camber. Insufficient tension produces loose, poorly wound coils that telescope or collapse.

### In-Process Quality Monitoring

Check strip width every 500–1,000 meters during a run:
- Measure 3–5 strips across the arbor at each check point
- Flag any strip drifting beyond ±50% of the width tolerance — investigate for spacer migration or uneven clamping
- Monitor burr height trend — increasing burr indicates knife wear approaching regrind threshold

---

## Step 6: Coil Changeover

### Sequence

1. Decelerate to threading speed before the coil end
2. Stop the line before the coil end reaches the straightener — avoid threading the bare mandrel core
3. Unload finished slit coils from the recoiler
4. Clear residual strip from the line if changing material grade or width
5. Load the new master coil
6. If arbor setup is unchanged (same width and clearance), thread and resume
7. If setup changes, reconfigure the arbor before threading

### Changeover Time Benchmarks

| Setup Condition | Manual Changeover | With Cassette Tooling |
|-----------------|------------------|----------------------|
| Same width, same material | 10–20 min | 5–10 min |
| Same width, different gauge | 20–35 min | 10–15 min |
| New width, same material | 45–90 min | 15–25 min |
| Full tooling change | 90–150 min | 25–40 min |

## Quality Parameters and Inspection

### Burr Height

The most common quality parameter for slit strip. Measured with a contact profilometer or optical comparator. Acceptable burr height depends on downstream application:

- Automotive stamping: ≤ 10% of material thickness
- Electrical laminations: ≤ 5% of material thickness
- Battery electrode: ≤ 2% of material thickness (often expressed in absolute μm)

### Edge Rollover (Rollover Depth)

The depth of material that deforms downward before fracture. Rollover is acceptable in controlled amounts — typically 20–40% of thickness. Excessive rollover indicates clearance too wide or knife faces worn.

### Strip Camber

Lateral curvature of the slit strip. Caused by differential elongation across the strip width — typically from uneven tension, misaligned knife pairs, or material anisotropy. Specification: typically ≤ 1 mm per meter of strip length for precision applications.

### Width Tolerance

Measured across 5+ points per coil, comparing to work order specification. Track variation across the arbor (strip-to-strip) and along the coil (front-to-back) separately. Strip-to-strip variation indicates spacer accuracy; along-coil variation indicates tension instability or material width variation in the master coil.

## Common Defects and Root Causes

| Defect | Most Likely Cause | Corrective Action |
|--------|------------------|-------------------|
| Excessive burr (one side) | Clearance too small on one knife pair | Recheck individual knife clearance; regrind knife if edge is worn |
| Excessive burr (all strips) | Clearance too small overall | Recalculate and reset horizontal clearance |
| Torn / ragged edge | Clearance too large | Reduce clearance to specification |
| Strip camber | Uneven tension across arbor; coil camber | Check tension distribution; compensate for material camber |
| Width variation along coil | Tension instability; straightener setting | Verify loop stability; check straightener roll gap |
| Strip wrapping on arbor | Stripper rings worn or undersized | Replace stripper rings; check ring OD vs knife OD |
| Knife chipping (carbide) | Arbor runout; spacer tolerance error | Measure arbor runout; replace worn spacers |

## Process Optimization

### Clearance Optimization

For a new material or knife combination, run a clearance sensitivity test: slit 5 test passes at clearances of −10%, specified, and +10% of nominal. Compare burr height, rollover, and edge fracture zone at each setting. The optimum clearance minimizes burr height while maintaining acceptable rollover depth.

### Knife Life Extension

- Never run past the regrind trigger point — micro-chipping beyond the trigger accelerates to macro-chipping
- Log knife life by material processed (meters or tonnes) per arbor position — some positions cut more total material than others and wear faster
- Rotate knife position assignments periodically to equalize wear across the knife set

### Throughput vs. Quality Trade-offs

Higher line speed increases production rate but can increase edge temperature, change clearance geometry through thermal expansion, and reduce time for loop control to respond to tension changes. Qualify maximum line speed for each material/thickness combination by monitoring burr height and width tolerance at stepped speed increments.

**Q: How do you calculate knife clearance?**
As a percentage of material thickness: 8–12% for mild steel; 10–15% for stainless; 5–8% for aluminum; 4–6% for copper. Example: 1.5 mm stainless at 12% = 0.18 mm.

**Q: What causes burr on slit edges?**
Clearance too small (most common), worn knife edge, or misaligned knife pair. Burr on one side = individual knife pair issue. Burr across all strips = overall clearance setting.

**Q: How do you set up a new width?**
Calculate spacer widths → load knife/spacer stack → set horizontal clearance → set penetration depth → clamp to torque spec → thread at slow speed → verify widths before production speed.

**Q: What causes strip camber?**
Uneven tension across the arbor, misaligned knife pairs, or material anisotropy in the master coil. Controlled through uniform tension distribution and pre-checking coil straightness.
