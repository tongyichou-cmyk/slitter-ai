# 分條線刀具組裝實務：操作員的完整安裝步驟與換刀流程

分條機的換刀作業，是每個操作員都必須熟練的核心技能。做得好，20–30 分鐘完成換刀、切口品質穩定；做得不好，換刀耗費 2 小時還出現各種品質問題。

本文提供完整的刀具組裝實務指南，從工具準備到安裝驗證，逐步說明。

---

## 換刀作業前的準備

### 安全確認

**換刀是停機作業，務必在開始前完成以下安全確認：**

- 確認機台已完全停止，主電源已鎖定（Lockout / Tagout）
- 確認刀軸已完全靜止
- 配戴適當防護裝備：防割手套、安全鞋、護目鏡
- 確認工作區域無其他人員進入

### 工具與材料準備

**必要工具：**
- 千分尺（測量刀片與間隔環厚度）
- 液壓泵（如使用液壓螺母）
- 清潔工具：布料、清潔劑、防鏽油
- 記錄表（記錄安裝規格與順序）

**材料確認：**
- 依換刀單確認刀片型號、數量
- 確認間隔環規格與片數
- 準備新批次的刀具時，逐片量測並記錄實際尺寸

## 拆刀步驟

### 第一步：鬆開鎖緊件

依機台設計，鬆開刀軸末端的鎖緊件：

- **液壓螺母**：連接液壓泵，加壓至指定壓力，旋鬆螺母
- **機械螺母**：使用扭力扳手逐步鬆開，注意螺紋方向（部分刀軸為左旋螺紋）

### 第二步：取出刀具組

依序從刀軸上取下：末端鎖緊件 → 末端間隔環 → 刀片組（刀片 + 間隔環 + 剪切環）→ 基準端間隔環

**注意：** 取下的刀具組應放置在專用的刀具盤或軟墊上，刀刃勿與硬物直接碰撞。

### 第三步：清潔刀軸

1. 以清潔布擦拭刀軸表面，去除油污與金屬粉末
2. 確認刀軸肩部（基準面）無磨損或異物
3. 確認刀軸螺紋完好無損
4. 在刀軸表面薄塗一層防鏽油

## 裝刀步驟

### 第一步：確認換刀規格

依換刀單確認：
- 成品寬度規格
- 各間隔環的排列順序與厚度
- 刀片的安裝方向（上下刀的配對方式）

> **刀片安裝方向原則：** 上下刀片必須以「剪刀式」咬合——上刀片右側面與下刀片左側面配對。安裝前確認刀片的刀刃方向，不可裝反。

### 第二步：量測刀具確認

安裝前，對每片刀具進行量測確認（特別是新品或送磨後回來的刀具）：

| 量測項目 | 工具 | 允許公差 |
|---------|------|---------|
| 刀片厚度 | 千分尺 | ±0.002mm（高精度 ±0.001mm） |
| 間隔環厚度 | 千分尺 | ±0.002mm（高精度 ±0.001mm） |
| 刀片外徑一致性 | 外徑千分尺 | 同組允差 ±0.002mm |

### 第三步：安裝刀具組

依下列順序，將刀具組套入刀軸：

```
刀軸固定端（肩部） → 基準間隔環 → [刀片 + 間隔環 + 剪切環 × n道] → 末端間隔環 → 鎖緊件
```

**安裝原則：**
1. 一次套入一個零件，確認每片零件都完全平貼（無傾斜或懸空）
2. 大間隔環靠近刀片：刀片兩側的間隔環選用最厚的，有助於剛性
3. 剪切環安裝方向：橡膠外徑稍大於刀片外徑，確認朝向正確方向
4. 用手推送零件時，**戴防割手套**，避免刀刃割傷

### 第四步：鎖緊

- **液壓螺母**：連接液壓泵，緩慢加壓至製造商規定的操作壓力，維持確認無洩漏後關閉
- **機械螺母**：以扭力扳手按規定扭矩鎖緊，分兩次進行（先 50% 扭矩，再 100% 扭矩）

## 安裝後驗證

### 目視確認
- 確認所有刀片、間隔環、剪切環排列整齊
- 手轉刀軸一圈，確認無卡頓、異常阻力或異音
- 確認上下刀軸的刀片咬合方向正確

### 試切驗證

1. 先以試料（廢料或邊料）進行試切
2. 量測各條成品的實際寬度，確認是否在公差範圍內
3. 在帶料長度方向取三個點量測，確認寬度一致性
4. 目視確認各道切口毛邊大小均勻
5. 試切確認後，記錄換刀完成時間與確認結果

## 縮短換刀時間的實務技巧

**技巧一：刀具組預備（Pre-staging）**
在前一個規格還在生產時，就開始準備下一個規格的刀具組。

**技巧二：刀具模組化**
將常用規格的刀具組預先組裝好放在刀具盤上，換刀時直接整組交換。

**技巧三：標準化刀具管理**
- 刀軸、刀片、間隔環都標上識別碼
- 建立常用規格的「換刀配方」（Tooling Recipe），記錄每種規格的間隔環排列順序
- 定期訓練操作員，統一換刀 SOP

**技巧四：輕量間隔環**
使用輕量型間隔環，減少換刀體力負擔，縮短安裝時間。

## 常見安裝錯誤與預防

| 錯誤 | 後果 | 預防方法 |
|------|------|---------|
| 刀片安裝方向錯誤 | 無法切割或刀片損壞 | 安裝前確認刀刃方向，設計明確的方向標記 |
| 間隔環排列順序錯誤 | 成品寬度偏差 | 使用換刀配方記錄，安裝完成後逐一核對 |
| 刀軸未清潔 | 精度誤差、刀具鬆動 | 每次換刀必清潔，列為 SOP 強制步驟 |
| 鎖緊力不足 | 刀具在生產中鬆動，切口品質劣化 | 使用扭力工具或液壓螺母確保一致夾緊力 |
| 刀片未量測就安裝 | 公差不符導致品質問題 | 新刀片和送磨回的刀片，安裝前一律量測 |
| 剪切環漏裝 | 帶料纏繞刀軸 | 安裝完成後對照清單逐一確認 |

**Q：試切第一刀切口品質特別差，正常嗎？**
正常，屬於「磨合」效應。建議用 3–5 米試料確認後再正式生產，取中段樣品確認品質。

**Q：可以只換其中一軸嗎？**
原則上上下刀片應成組管理，避免磨耗差異造成切口不均。只換一軸時，確認新舊刀片外徑差異在允許範圍內。

**Q：換刀後刀軸溫度升高很快，正常嗎？**
適度溫升正常。若溫度異常快速升高或燙手，可能是間隙過小或刀片安裝方向錯誤，需立即停機檢查。
