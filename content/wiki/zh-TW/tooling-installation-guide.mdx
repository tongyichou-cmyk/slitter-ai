# 分條線刀具組裝實務：完整安裝步驟、換刀流程與常見錯誤防範

分條機的刀具安裝，看起來只是把刀片和間隔環套在刀軸上、鎖緊螺母，但實際上是一個精密的組裝工程。安裝中的每一個細節——刀軸的清潔程度、間隔環的實際尺寸確認、刀片的安裝順序、螺母的夾緊力——都直接決定帶料的寬度精度和切口品質。

一次不規範的安裝，可能導致一整卷料的損失；而一套標準化的換刀流程，可以將換刀時間從 90 分鐘縮短到 25 分鐘以內，同時確保每次安裝的品質一致。

本文提供完整的刀具安裝標準作業程序（SOP），適合操作員和工程師共同使用。

---

## 換刀前的準備工作

### 材料確認

在開始換刀前，確認：

1. **工作指令：** 確認本次切割的帶料寬度規格、材料種類、厚度
2. **刀具狀態：** 確認準備使用的刀片是否已研磨到位，量測記錄是否完整
3. **間隔環選擇：** 根據寬度規格，計算各段間隔環組合，用量測記錄確認每片間隔環的實際厚度

### 工具準備

安裝開始前，準備以下工具：

| 工具 | 用途 |
|------|------|
| 外徑千分尺（0.001mm 分度值） | 量測刀片外徑、間隔環厚度 |
| 千分表 + 磁性表座 | 安裝後量測刀軸跳動量 |
| 無塵布 + 清潔劑 | 清潔刀軸和零件 |
| 液壓泵（如使用液壓螺母） | 夾緊刀組 |
| 力矩扳手（如使用傳統螺母） | 按規定扭矩鎖緊 |
| 放大鏡（10×） | 檢查刀刃狀態 |
| 安裝輔助夾具（如有） | 刀片搬運和定位 |

## 拆刀步驟

### 步驟 1：停機並確認安全

1. 確認分條機已完全停止，主電源已切斷或鎖定（Lock Out Tag Out）
2. 等待刀軸完全靜止，不能用手或工具強制制動

### 步驟 2：鬆開夾緊機構

**液壓螺母：**
1. 連接液壓泵到液壓螺母的注油口
2. 開啟洩壓閥，緩慢釋放油腔壓力（不要瞬間全開）
3. 待壓力歸零後，手力旋開液壓螺母

**傳統螺母：**
1. 使用大扭矩工具鬆開螺母（逆時針）
2. 鬆開時注意刀組的重量支撐，防止突然掉落

### 步驟 3：記錄刀組狀態

在拆卸前，用手機或相機記錄現有刀組的配置，包括：
- 間隔環的排列順序和方向（有非對稱件時尤其重要）
- 剪切環的位置
- 特殊零件的位置

### 步驟 4：依序拆卸零件

按照以下順序（與安裝順序相反）拆卸：
1. 螺母 → 2. 末端間隔環 → 3. 刀片/間隔環交替拆除（從末端向基準端）→ 4. 剪切環

**拆下的零件按順序排列在工作台上，不要混雜。** 若計劃重複使用相同配置，按拆下的順序排列，安裝時直接反向組裝。

## 裝刀步驟

### 步驟 1：清潔刀軸

**這是最重要的一步，也是最常被忽略的一步。**

用無塵布蘸清潔劑，徹底清潔刀軸的：
- 安裝段表面（安裝刀片/間隔環的區域）
- 螺紋（清除金屬屑和舊螺紋膠）
- 基準面（刀組的定位基準）

清潔後用無塵布擦乾，確認無任何污染物。

**注意：** 刀軸表面上 0.01mm 的污染物或殘留切屑，都會被間隔環壓緊，形成等效的局部厚度增量，導致刀組定位誤差。

### 步驟 2：量測確認所有零件

安裝前，用千分尺逐片量測並記錄：

| 零件 | 量測項目 | 記錄目的 |
|------|---------|---------|
| 每片刀片 | 外徑、厚度 | 確認配對刀片外徑差 ≤ 0.002mm |
| 每片間隔環 | 厚度（量測三點取平均） | 計算累積總厚度，確認寬度設定 |
| 剪切環 | 外徑（確認超出量） | 確認是否在有效超出量範圍內 |

計算間隔環累積總厚度，對比工作指令的目標寬度，確認無誤後才開始安裝。

### 步驟 3：安裝刀組（上刀軸、下刀軸各自安裝）

**安裝順序（從基準端開始）：**

```
基準定位環 → 第一組（大間隔環 + 刀片 + 大間隔環）→ 第二組 → ... → 末端間隔環 → 螺母/液壓螺母
```

**安裝技巧：**

**1. 大件優先：** 每個刀位用盡量少的間隔環片數（大厚度優先），減少累積誤差來源。

**2. 刀片方向確認：** 確認刀片的切割方向。部分刀片（尤其是斜面側面刀片）有安裝方向，上刀和下刀的方向要對應。

**3. 均勻施力：** 每片零件套入刀軸時，確認均勻貼合，無偏斜。刀片或間隔環如果歪斜安裝，會造成局部接觸應力，影響安裝精度。

**4. 剪切環位置：** 剪切環安裝在刀片的外側（離帶料接觸面的外側），確認超出量在規格內。

### 步驟 4：鎖緊夾緊機構

**液壓螺母：**
1. 手力旋入液壓螺母到底
2. 確認螺母端面均勻接觸刀組端面（無偏斜）
3. 連接液壓泵，緩慢加壓至設備手冊規定的設定壓力
4. 關閉截止閥，斷開液壓泵

**傳統螺母：**
1. 手力旋入螺母到底
2. 使用力矩扳手，按規定力矩（查設備手冊）鎖緊
3. 分兩次鎖緊：第一次 80% 力矩，第二次達到全力矩

### 步驟 5：安裝後驗證

**這一步不能省略。**

**5-1：量測刀片跳動量（TIR）**

用千分表量測每片刀片的徑向跳動：

| 刀片材質 | 允許 TIR |
|---------|---------|
| 工具鋼（D2、HSS） | ≤ 0.005mm |
| 硬質合金 | ≤ 0.003mm |

超過允許值：重新拆下，清潔刀軸和刀片，確認無異物後重新安裝。

**5-2：確認間隙設定**

用千分表確認各刀對的水平間隙是否符合設定值，不要僅靠計算值。

**5-3：目視確認剪切環位置**

確認每個剪切環的超出量均勻，無偏斜。

## 穿料與試切

### 穿料步驟

1. 以低速（5–10m/min）穿料通過刀組
2. 確認每條帶料的走向正確，無交叉
3. 確認廢邊走向正確
4. 緩慢提速到 30–50m/min，試切 10–20 米

### 試切驗證

在試切段取樣，量測：
- 所有條帶的實際寬度（對比工作指令公差）
- 各道切口的毛邊高度
- 帶料邊緣是否有異常

確認全部符合規格後，加速到生產速度。

## 縮短換刀時間的實務技巧

### 技巧 1：預先組裝備用刀組

在現有刀組生產中，提前將下一個品規的刀組預組裝在備用刀軸上，完成間隔環量測和計算。到換刀時，只需交換刀軸，而不是從零開始組裝。

**節省時間：** 可縮短換刀時間 30–50%

### 技巧 2：建立間隔環套件（Spacer Kit）

對於常見的幾個寬度規格，預先組裝好專用的間隔環套件，並用色標或掛牌標識。到換規格時，直接取用對應套件，不需要每次重新計算和挑選間隔環。

### 技巧 3：量測表格標準化

建立標準的換刀量測記錄表，包含：
- 刀片外徑確認欄
- 間隔環累積厚度計算欄
- 安裝後跳動量記錄欄
- 試切寬度記錄欄

操作員按表填寫，確保每次換刀品質一致，同時留下可追溯的安裝記錄。

### 技巧 4：採用輕量型間隔環

重型刀組的搬運是換刀時間的重要佔比。採用輕量型間隔環，一組 20 片的刀組重量可降低 40–60%，顯著減少搬運時間和操作員體力消耗。

## 常見安裝錯誤

| 錯誤 | 後果 | 防範方法 |
|------|------|---------|
| 刀軸清潔不徹底 | 間隔環定位偏差，寬度超差 | 每次換刀都清潔，不要省略 |
| 依賴標稱值而非實測值 | 累積誤差超差 | 逐片量測，不信標稱值 |
| 刀片未配對確認 | 刀對外徑差過大，切口不均 | 配對量測外徑，差值 ≤ 0.002mm |
| 跳動量確認省略 | 週期性毛邊，影響品質 | 安裝後必量 TIR |
| 間隙設定只靠計算 | 實際間隙與設定值偏差 | 千分表確認實際間隙 |
| 螺母夾緊力不足 | 刀片位移，寬度漂移 | 按規定力矩或液壓壓力執行 |
| 剪切環安裝方向錯誤 | 帶料排斥效果差，容易纏繞 | 安裝前確認方向 |

## 安全注意事項

- **刀片的切割面極為鋒利**，安裝時必須使用防切割手套
- 刀片搬運時，避免讓刀刃面碰觸任何硬物（包括地面、工具台），防止崩刃
- 液壓系統操作時，不要超過最大允許壓力
- 大型刀組（外徑 > 200mm）搬運時，使用吊具或兩人協作，不要單人強行搬運

**Q：換刀後品質比之前差，最可能的原因？**
(1) 刀軸清潔不徹底；(2) 新刀外徑不同，間隙改變未重新確認；(3) 跳動量超差（未量測）。按序排查。

**Q：如何確認間隔環厚度準確？**
千分尺（0.001mm 分度值）在三點量測取平均。必須以實測值為準，不能信標稱值。

**Q：換刀時間怎麼縮到 30 分鐘以內？**
液壓螺母（-30 分鐘）+ 備用刀組預組裝（-20 分鐘）+ 輕量間隔環（-10 分鐘）= 可達 20–25 分鐘。

**Q：安裝後跳動量超差怎麼辦？**
拆下刀片，清潔刀軸安裝面，重新安裝後再量測。若仍超差，確認刀片內孔磨損狀態。
