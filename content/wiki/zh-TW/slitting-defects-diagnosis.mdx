# 分條加工缺陷完全診斷指南：毛邊、邊波、刀痕、帶料變形系統排查

分條生產線停機的代價很高，而大多數分條缺陷都有明確的根本原因。問題是：同樣的「毛邊」症狀，可能來自五種不同的根本原因，處理方式完全不同。

本文提供系統化的分條缺陷診斷框架，依缺陷類型分類，逐一說明症狀特徵、根本原因，以及從根本解決的方法。**正確診斷比立即換刀更重要。**

---

## 診斷的基本原則

在具體診斷之前，建立正確的排查邏輯：

1. **先確認機台精度，再排查刀具，最後調整製程參數。** 機台跳動過大的問題，換刀也解決不了。
2. **觀察缺陷的分佈規律。** 所有條帶都有？只有特定條帶？沿料卷長度方向週期性出現？這些模式直接指向不同的根本原因。
3. **一次只改一個變量。** 同時調整間隙和壓入量，你就不知道哪個改善有效。
4. **記錄基準數據。** 換新刀後立即記錄電流、寬度、毛邊高度，作為後續判斷磨耗的基準。

## 12 大缺陷類型系統診斷

---

### 缺陷 1：所有條帶均有飽滿連續毛邊

**症狀特徵：**
- 毛邊高度均勻，沿切割方向連續分佈
- 毛邊根部光滑，呈捲曲狀
- 切斷面撕裂帶比例過高

**根本原因：水平間隙過大**

**確認方法：** 量測現有間隔環，計算實際水平間隙，對照各材料建議值（見[間隙設定指南](/zh-TW/wiki/slitting-clearance-guide)）

**解決方案：**
1. 縮小水平間隙至建議值
2. 若間隔環已磨損，更換新間隔環重新設定

---

### 缺陷 2：所有條帶均有破碎絲狀毛邊

**症狀特徵：**
- 毛邊不連續，呈短絲、鬚狀或碎片狀
- 切斷面有亮帶（二次剪切痕跡）
- 主軸電流可能偏高

**根本原因：水平間隙過小，或刀片嚴重磨耗**

**確認方法：**
1. 先確認水平間隙設定，若間隙正確，再目視刀片磨耗狀態
2. 用放大鏡觀察刀刃，確認是否有明顯磨耗亮帶或崩缺

**解決方案：**
- 若間隙過小：擴大水平間隙
- 若刀片磨耗：安排研磨並重新設定間隙

---

### 缺陷 3：奇數/偶數條帶毛邊不同

**症狀特徵：**
- 第 1、3、5... 條帶毛邊大，第 2、4、6... 條帶正常（或反之）
- 或特定幾道條帶系統性毛邊偏大，其餘正常

**根本原因：刀軸基準點偏移，或上下刀軸不平行**

**確認方法：**
1. 量測第一道刀的間隙（首道基準點）
2. 確認上下刀軸的平行度

**解決方案：**
1. 重新確認刀軸首道定位基準，重設所有刀對間隙
2. 若刀軸平行度超差，需要機台調整

---

### 缺陷 4：同一條帶兩側毛邊不同

**症狀特徵：**
- 帶料左緣毛邊大，右緣小（或反之）
- 此現象在每條帶料上重複，但不同條帶的方向可能不同

**根本原因：特定刀對的間隙左右不對稱，或刀片側面磨耗不對稱**

**確認方法：**
1. 找出問題條帶對應的刀對
2. 量測該刀對的左右兩側間隙是否一致

**解決方案：**
- 重新調整該刀對的間隔環，確保兩側間隙對稱
- 若刀片側面磨耗不對稱，需要研磨恢復對稱性

---

### 缺陷 5：毛邊沿料卷長度方向週期性變化

**症狀特徵：**
- 切割初期毛邊正常
- 每隔固定長度（例如每圈或每半圈），毛邊高度明顯增大後恢復正常

**根本原因：刀軸或刀片有偏心（跳動量過大）**

**確認方法：**
用千分表量測刀片的徑向跳動（TIR）：
- 工具鋼刀片允許值：TIR ≤ 0.005mm
- 硬質合金刀片允許值：TIR ≤ 0.003mm

超過以上值即需要重新安裝或換刀。

**解決方案：**
1. 重新安裝刀片（確認內孔無異物，刀軸清潔）
2. 若重新安裝後仍超差，確認是刀片內孔磨損還是刀軸磨損

---

### 缺陷 6：越切越大的毛邊（同一卷料，從頭到尾漸增）

**症狀特徵：**
- 切割初期毛邊合格，切到中後段毛邊逐漸增大
- 不同卷料的規律相同

**根本原因：刀片正常磨耗，有效間隙隨使用時間增大**

**確認方法：** 這是正常磨耗特徵，不需要特別確認

**解決方案：**
- 記錄現有刀具的研磨觸發里程
- 建立主動研磨週期，在毛邊到達觸發值前安排研磨

---

### 缺陷 7：帶料邊緣出現邊波（Edge Wave）

**症狀特徵：**
- 帶料邊緣呈波浪狀，而非平直
- 通常在收卷後更明顯（波浪壓縮後呈現）

**可能原因（多個）：**

| 原因 | 確認方法 |
|------|---------|
| 收卷張力不均 | 確認各條帶張力設定 |
| 材料本身有殘餘應力 | 確認母卷材料是否有橫向應力不均 |
| 壓入量不足 | 確認壓入量設定 |
| 刀軸跳動過大 | 量測 TIR |

**解決方案（依原因）：**
- 張力不均：調整各條帶的獨立張力
- 材料問題：增加矯直器設定強度
- 壓入量不足：略微增加壓入量

---

### 缺陷 8：帶料出現縱向刀痕（Scratch Line）

**症狀特徵：**
- 帶料表面出現與切割方向平行的連續劃痕
- 劃痕位置固定，不在切口位置

**根本原因：刀組中有異物（金屬屑）或間隔環端面有缺損**

**確認方法：**
1. 停機，仔細檢查刀組，尋找鬆動的金屬碎片或異物
2. 確認間隔環端面是否有缺口或毛刺

**解決方案：**
1. 清除刀組異物
2. 更換有缺損的間隔環
3. 確認刀組安裝區域無切屑積累

---

### 缺陷 9：帶料寬度偏差（寬度超出公差）

**症狀特徵：**
- 成品帶料寬度系統性偏大或偏小
- 或不同條帶的寬度差異超過公差

**可能原因：**

| 症狀 | 可能原因 |
|------|---------|
| 所有條帶系統性偏寬 | 間隔環累積尺寸偏差，總寬度偏大 |
| 所有條帶系統性偏窄 | 間隔環累積尺寸偏差，總寬度偏小 |
| 特定條帶偏寬，另一條帶對應偏窄 | 局部間隔環厚度錯誤 |
| 同一條帶沿長度方向寬度波動 | 刀片位置在切割中漂移（鎖緊力不足）|

**解決方案：**
- 逐片量測間隔環實際厚度，重新計算並確認累積尺寸
- 若是刀片位置漂移：確認液壓螺母夾緊力是否足夠

---

### 缺陷 10：帶料纏繞刀軸

**症狀特徵：**
- 切割後帶料不能順暢脫離刀面，纏繞在上刀或下刀軸上
- 觸發急停或刀軸損傷

**可能原因：**
- 剪切環磨耗過大，超出量不足
- 剪切環 Shore A 硬度不足（高速下開花）
- 薄料靜電吸附（需靜電消除設備）
- 材料本身黏性（軟鋁、純銅）需要塗層刀具

**解決方案：**
- 更換剪切環，確認超出量和硬度符合規格
- 超薄料或高速線考慮增加靜電消除棒

---

### 缺陷 11：切割中硬質合金刀片崩刃

**症狀特徵：**
- 切割中突然出現局部毛邊急增、或帶料邊緣出現缺口
- 停機後發現某片刀片有明顯缺口

**根本原因（幾乎都是安裝問題，不是刀片本身）：**

| 原因 | 機率 |
|------|------|
| 刀軸跳動超過 0.003mm | 最高 |
| 間隔環公差過大，刀對未對準 | 高 |
| 間隙設定過小，刀刃承受過大壓縮力 | 高 |
| 材料有焊縫或硬點，衝擊脆性刀片 | 中 |
| 刀片本身有隱裂（少見） | 低 |

**解決方案：**
1. 立即停機，更換崩缺刀片
2. 重新量測刀軸跳動
3. 確認間隔環精度和間隙設定
4. 確認是否有焊縫通過刀組的風險（如需要，在焊縫前降速）

---

### 缺陷 12：帶料寬度沿長度方向漂移（越切越寬或越切越窄）

**症狀特徵：**
- 帶料寬度在切割過程中緩慢漂移，不是突然跳變

**根本原因：刀片在切割中因熱膨脹或鎖緊力不足而位移**

**確認方法：**
1. 停機後量測刀片位置，確認是否已偏移
2. 確認液壓螺母或刀軸螺母的夾緊力

**解決方案：**
- 確認夾緊力符合規定值
- 對於熱膨脹問題：在切割前讓機台預熱 10–15 分鐘達到熱平衡

## 系統排查流程圖

```
觀察缺陷分佈
    │
    ├─ 所有條帶均有 → 間隙設定問題 / 刀片整體磨耗
    │
    ├─ 特定條帶有 → 特定刀對問題
    │
    ├─ 沿長度方向週期性 → 刀軸/刀片跳動
    │
    ├─ 沿長度方向漸進性 → 磨耗 / 熱膨脹 / 鎖緊力不足
    │
    └─ 隨機分佈 → 材料問題 / 異物

→ 確認機台精度（跳動、平行度）
→ 確認刀具狀態（磨耗、崩缺、外徑公差）
→ 確認製程參數（間隙、壓入量、夾緊力）
→ 一次改一個變量，試切驗證
```

## 缺陷記錄與趨勢分析

建立每個班次的缺陷記錄，追蹤以下指標：
- 毛邊高度（每 500m 取樣）
- 帶料寬度（每 500m 取樣）
- 主軸電流
- 換刀/研磨事件

定期分析趨勢，可以：
1. 預測刀具壽命，提前安排研磨
2. 識別特定材料批次的異常（材料硬度偏差）
3. 評估不同刀具品牌/批次的實際壽命差異

**Q：出現毛邊，第一步查什麼？**
先看缺陷分佈：所有條帶均有 → 查間隙設定；特定條帶 → 查對應刀對；週期性 → 量測跳動量。80% 毛邊問題是間隙問題，不是刀具問題。

**Q：硬質合金刀片一直崩刃，是刀片品質差嗎？**
幾乎都不是。先量測刀軸跳動（應 ≤ 0.003mm），再查間隔環精度和間隙設定。

**Q：換新刀後品質反而變差？**
新刀外徑改變，壓入量和有效間隙都會改變。換新刀後必須重新確認並調整間隙設定。
