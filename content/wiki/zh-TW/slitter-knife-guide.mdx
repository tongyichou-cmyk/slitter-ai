# 分條刀完全指南：材質、幾何規格與選購邏輯

在金屬分條加工中，刀片是整條生產線最直接決定切口品質的零件。一把規格正確、材質適當的分條刀，能讓你的切口乾淨、毛邊極小、刀片壽命長；反之，一把錯誤的刀具選擇，會讓你在毛邊、換刀停機、研磨費用上持續消耗。

本文將從材質、幾何規格、選購邏輯到更換時機，完整說明分條刀的選用知識。

---

## 什麼是分條刀？

**分條刀**（Slitter Knife，又稱圓形刀片、Circular Slitting Blade）是金屬分條機刀組（Slitter Head）的核心零件。上下兩片成對的圓形刀片，像剪刀一樣協同運作，對進給中的金屬帶料進行縱向剪切。

每次切割，刀片都承受極大的剪切應力、摩擦熱與疲勞循環。選刀的本質，就是在**刀具硬度（耐磨）**與**韌性（抗崩刃）**之間找到最適合你材料的平衡點。

---

## 分條刀的主要材質

### D2 工具鋼（最常見）

D2 是業界最廣泛使用的分條刀材質，屬於高碳高鉻冷作工具鋼。

| 特性 | 說明 |
|------|------|
| 硬度 | HRC 58–62 |
| 耐磨性 | 優 |
| 韌性 | 中等 |
| 適用材料 | 冷軋鋼、熱軋鋼、鍍鋅鋼、一般不鏽鋼 |
| 優點 | 性價比高、研磨性好、泛用性強 |
| 缺點 | 不適合高硬度材料（AHSS、高矽鋼） |

D2 是大多數一般金屬加工廠的主力選擇，尤其適合厚度 0.3mm 至 6mm 的碳鋼與鍍鋅鋼。

---

### M2 高速鋼

M2 屬於鎢鉬系高速鋼，熱硬性更優，適合切割速度較高的場景。

| 特性 | 說明 |
|------|------|
| 硬度 | HRC 62–65 |
| 熱穩定性 | 優（高速切割時維持硬度） |
| 適用材料 | 薄材、高速分條線、不鏽鋼 |
| 缺點 | 成本高於 D2，韌性略低 |

如果你的生產線速度超過 200m/min，或切割薄料（< 0.5mm）時刀片容易過熱，M2 是值得升級的選擇。

---

### 硬質合金（Carbide）

硬質合金（Cemented Carbide，俗稱鎢鋼）是耐磨性最強的刀具材質，硬度可達 HRA 89–93，遠超工具鋼。

| 特性 | 說明 |
|------|------|
| 硬度 | HRA 89–93 |
| 耐磨性 | 極優 |
| 韌性 | 較低（不耐衝擊） |
| 適用材料 | 矽鋼片（電磁鋼）、銅帶、鋁箔、AHSS 高強鋼 |
| 優點 | 壽命可達工具鋼的 5–10 倍 |
| 缺點 | 單片成本高，且不耐振動與衝擊 |

在切割矽鋼片、高強度汽車鋼（AHSS/UHSS）或銅箔等精密材料時，硬質合金刀片雖然初始成本高，但綜合壽命與品質往往是最經濟的選擇。

---

### 粉末冶金鋼（Powder Metal）

粉末冶金鋼是介於高速鋼與硬質合金之間的選項，韌性比硬質合金好，但耐磨性高於一般工具鋼。

| 特性 | 說明 |
|------|------|
| 硬度 | HRC 63–67 |
| 耐磨性 | 高 |
| 韌性 | 中高（比硬質合金好） |
| 適用材料 | 高硬度不鏽鋼、雙相鋼、HSLA 高強低合金鋼 |

對於切割雙相不鏽鋼或 HSLA 鋼，PM 鋼往往是比硬質合金更穩定的選擇，因為不易崩刃。

---

## 分條刀的幾何規格解讀

### 外徑（OD）
決定刀片在刀軸上的安裝位置與切割半徑。常見外徑範圍從 75mm 到 250mm 不等，依機台規格而定。

**重要原則：** 同一刀組內，所有刀片的外徑必須相互匹配（誤差控制在 ±0.002mm 以內），否則各道切口的下壓量不一致，導致毛邊大小不均。

### 內徑（ID）
刀片套入刀軸的孔徑，必須與刀軸直徑精確配合。間隙過大會造成刀片偏擺，影響切割直線度。

### 厚度（T）
刀片厚度決定切割刀距的基礎間距，與間隔環搭配使用。厚度公差一般要求 ±0.002mm（高精度場合 ±0.001mm）。

## 各材料對應的刀具選擇建議

| 切割材料 | 建議刀具材質 | 說明 |
|---------|------------|------|
| 冷軋鋼（CR） | D2 | 標準選擇，性價比最高 |
| 熱軋鋼（HR） | D2 | 厚料建議選較高韌性等級 |
| 鍍鋅鋼（GI） | D2 / M2 | 鋅塗層會加速磨耗，M2 壽命更長 |
| 不鏽鋼（SUS） | M2 / PM 鋼 | 硬化不鏽鋼考慮 PM 鋼或硬質合金 |
| AHSS / UHSS 高強鋼 | PM 鋼 / 硬質合金 | 普通工具鋼壽命極短 |
| 矽鋼片（電磁鋼） | 硬質合金 | 磁性要求嚴格，切口品質至關重要 |
| 鋁合金 | D2 / 硬質合金 | 薄鋁箔優先硬質合金 |
| 銅 / 銅合金 | D2 / 硬質合金 | 銅箔需硬質合金 |

## 如何判斷分條刀需要更換或研磨？

**切口品質信號：**
- 切口毛邊（Burr）高度超過材料厚度的 10%
- 切口邊緣出現撕裂而非剪切形貌
- 切割後帶料出現邊波（Edge Wave）

**設備信號：**
- 分條機主軸電流或扭矩異常升高
- 切割時出現異常振動或噪音

**外觀信號：**
- 刀刃邊緣出現明顯缺口或崩刃
- 刀刃面出現亮帶（摩擦面積增加）

> **原則：** 不要等刀片完全失效才研磨。早期研磨的去除量少、刀片可多次重磨；拖到嚴重磨耗再磨，每次研磨量大，刀片壽命大幅縮短。

**Q：D2 和 M2 刀片有什麼實際差異？**
D2 性價比高、泛用性強，適合一般碳鋼；M2 熱穩定性更好，適合高速分條或薄料切割。線速低於 150m/min 切割普通碳鋼，D2 已足夠。

**Q：硬質合金刀片是否值得投資？**
切割矽鋼片、AHSS、銅鋁箔時，硬質合金壽命達工具鋼 5–10 倍，綜合成本更低。切割一般碳鋼則成本優勢不明顯。

**Q：分條刀可以研磨幾次？**
取決於初始厚度與每次研磨去除量。初始 20mm 的刀片、每次去除 0.1mm，理論上可重磨數十次。早期研磨可最大化重磨次數。
