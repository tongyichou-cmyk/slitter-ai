# 分條刀完全指南：材質、幾何規格、選購與研磨實務

在金屬分條加工中，刀片是整條生產線最直接決定切口品質的零件。一把規格正確、材質適當的分條刀，能讓你的切口乾淨、毛邊極小、刀片壽命長；反之，錯誤的刀具選擇會讓你持續消耗在毛邊重工、換刀停機和不必要的研磨費用上。

本文從材質、幾何規格、選購邏輯到研磨實務，提供分條刀的完整技術知識。無論你是第一次規劃分條線刀具，還是想優化現有生產成本，都能從本文找到具體依據。

---

## 什麼是分條刀？

**分條刀**（Slitter Knife，又稱圓形刀片、Circular Slitting Blade）是金屬分條機刀組（Slitter Head）的核心零件。上下兩片成對的圓形刀片，像剪刀一樣協同運作，對進給中的金屬帶料進行縱向剪切。

分條切割的本質是**剪切（Shear Cutting）**，不是鋸切或磨削。材料在刀刃的壓應力與拉應力共同作用下，在切割線上發生可控斷裂。這個斷裂過程的品質——毛邊高度、塌角深度、斷面形貌——完全取決於刀具的幾何精度、材質硬度，以及間隙設定是否正確。

每次切割，刀片都承受極大的剪切應力、摩擦熱與疲勞循環。選刀的本質，就是在**刀具硬度（耐磨）**與**韌性（抗崩刃）**之間找到最適合你材料與產量要求的平衡點。

## 分條刀的主要材質

### D2 工具鋼

D2 是高碳高鉻冷作工具鋼（1.5% C、12% Cr），業界使用最廣泛的分條刀材質。正確熱處理後硬度達 HRC 58–62。

**性能特性：**
- 耐磨性優異，適合中高產量的一般金屬分條
- 研磨性良好，可用標準砂輪進行研磨維護
- 成本在精密刀具中屬中低段
- 韌性中等，在有夾雜物或軋製缺陷的材料上可能崩刃

**最適切割材料：** 冷軋鋼（CR）、熱軋鋼（HR）、鍍鋅鋼（GI）、預塗裝鋼、一般不鏽鋼（薄料）

**不適用場景：** 高強度鋼（AHSS/UHSS）、矽鋼片、高速線（＞200m/min）

---

### HSS M2 高速鋼

M2 為鎢鉬系高速鋼（6% W、5% Mo、2% V、4% Cr），熱處理後硬度 HRC 62–64。相較於 D2，M2 的優勢在於**熱硬性**——在高切割速度導致刀刃溫度升高時，仍能維持硬度，不會發生刃口軟化。

**性能特性：**
- 熱穩定性優於 D2，適合高速分條線
- 韌性高，在有焊縫或材料硬點的情況下比硬質合金安全
- 可多次研磨，每次研磨去除量小於硬質合金
- 成本高於 D2，但低於 M42 和硬質合金

**最適切割材料：** 高速線薄料（< 0.8mm）、一般不鏽鋼（304/316）、中高硬度碳鋼

---

### HSS M42 高速鋼（含鈷）

M42 在 M2 的基礎上加入 8% 鈷，進一步提升熱硬性。硬度達 HRC 65–67，在所有高速鋼等級中耐磨性最強。

**性能特性：**
- 比 M2 壽命提升 20–40%，尤其在切割不鏽鋼和高合金鋼時效果顯著
- 仍保有高速鋼的韌性，崩刃風險低於硬質合金
- 成本為 M2 的 1.2–1.4 倍

**最適切割材料：** 奧氏體不鏽鋼（304/316）、高合金鋼、切割速度較高的生產線

---

### 硬質合金（Tungsten Carbide）

硬質合金（碳化鎢鈷燒結體，WC-Co）是耐磨性最強的刀具材質，硬度 HRA 88–92（相當於 HRC 70–75）。本質上不是鋼，而是以鈷作為結合劑的金屬陶瓷複合材料。

**性能特性：**
- 耐磨性比 HSS 高 3–8 倍
- 極適合高磨耗材料（矽鋼片、銅箔、不鏽鋼箔）
- 韌性低——對安裝精度要求高，刀軸跳動超過 0.003mm 即可能崩刃
- 研磨需用金剛石砂輪，不能用一般研磨機
- 單片成本為 D2 的 4–6 倍，但壽命長，綜合成本通常更低

**最適切割材料：** 矽鋼片（電磁鋼）、銅箔、電池極片、AHSS 高強鋼、高產量不鏽鋼線

**成本分析舉例（以不鏽鋼 304、1.5mm、月產 200,000m）：**

| 指標 | HSS M2 | HSS M42 | 硬質合金 |
|------|--------|---------|---------|
| 每套刀具費用 | USD 120 | USD 180 | USD 550 |
| 壽命（米/套） | 40,000 | 60,000 | 200,000 |
| 月需套數 | 5 | 3.3 | 1 |
| 月刀具費 | USD 600 | USD 594 | USD 550 |
| 換刀次數/月 | 5 | 3.3 | 1 |
| 換刀人工費 | USD 150 | USD 99 | USD 30 |
| **月總成本** | **USD 750** | **USD 693** | **USD 580** |

硬質合金在高產量場景下，月總成本比 M2 低約 23%。

## 分條刀的幾何規格解讀

### 外徑（OD）

刀片外徑決定安裝位置與切割半徑。常見範圍 75–400mm，依機台規格而定。

**關鍵要求：** 同一刀組內，所有刀片的外徑誤差必須控制在 ±0.002mm 以內。一組 20 片刀軸中，若外徑誤差達 ±0.01mm，各刀對的壓入量可能差異 0.2mm，導致各道切口毛邊高度不均。

### 內徑（ID）/ 孔徑

刀片套入刀軸的孔徑，與刀軸採用 H7/h6 或 H6/h5 配合公差。孔徑同心度（相對於外徑）須在 0.002mm 以內，否則旋轉時產生偏擺，造成切割力不穩定。

### 厚度（T）

刀片厚度是刀軸排列間距的基準參數，與間隔環（Spacer）搭配決定各條帶料的分割寬度。

**精度要求：**
- 一般應用：±0.002mm
- 精密應用（電池、矽鋼片）：±0.001mm

一組 20 片刀的刀組，若每片厚度誤差 ±0.01mm，理論上累積誤差可達 ±0.2mm——這正是許多用戶帶料寬度不穩定的根本原因。

### 切刃面角度（Side Relief Angle）

刀片側面的研磨角度影響切割力與帶料側面品質：
- **直角側面（90°）：** 標準配置，適合大多數金屬分條
- **斜面側面（2–5°）：** 減少刀面與帶料的側向摩擦，用於容易黏刀的純鋁和銅材

## 刀具間隙設定：最關鍵的製程參數

**水平間隙（Horizontal Clearance）**是上下刀片切刃之間的橫向間距，以材料厚度的百分比表示，是分條切割品質最關鍵的製程參數。

### 各材料推薦間隙

| 材料種類 | 推薦間隙（% 厚度） |
|---------|-----------------|
| 冷軋鋼（CR）、熱軋鋼（HR） | 8–12% |
| 鍍鋅鋼、預塗裝鋼 | 10–14% |
| 不鏽鋼 304/316 | 10–15% |
| 馬氏體不鏽鋼（420/440） | 12–16% |
| 鋁合金 | 5–8% |
| 純銅、銅合金 | 4–6% |
| 矽鋼片（電磁鋼） | 6–10% |
| 高強鋼（AHSS/UHSS） | 8–12% |
| BOPP/PET 薄膜 | 1–3% |

### 間隙設定錯誤的後果

**間隙過小（＜最低建議值）：**
- 帶料下緣出現捲曲毛邊（Rolled Burr）
- 刀片承受過大壓縮力，磨耗加速
- 切割力升高，主軸電流增加

**間隙過大（＞最大建議值）：**
- 切斷面出現大面積撕裂區（Fracture Zone 比例上升）
- 毛邊高度增加，塌角加深
- 帶料邊緣出現波浪狀缺陷

**計算範例：**
切割 1.5mm 不鏽鋼 304，採用 12% 間隙：
```
水平間隙 = 1.5mm × 12% = 0.18mm（每對刀片）
```

## 壓入量（Penetration Depth）設定

壓入量是上刀片相對於下刀片在垂直方向的重疊量（Overlap）。

| 材料厚度 | 建議壓入量 |
|---------|----------|
| ＜ 0.5mm | 0.05–0.15mm |
| 0.5–2mm | 0.1–0.4mm |
| 2–5mm | 0.3–1.0mm |
| ＞ 5mm | 0.8–2.5mm |

壓入量不足：材料無法完全剪斷，出現撕裂或拉斷。
壓入量過大：切割力急增，刀片承受過大撓曲，邊緣品質惡化。

## 刀具安裝注意事項

### 刀軸跳動（Runout）檢查

刀軸安裝後，必須用千分表量測刀片外徑的徑向跳動（TIR）：
- 工具鋼刀片：TIR ≤ 0.005mm
- 硬質合金刀片：TIR ≤ 0.003mm（超過此值即可能崩刃）

### 間隔環精度匹配

刀片精度 ±0.002mm，間隔環必須達到相同精度，否則精密刀片的優勢被低精度間隔環抵消。

### 上緊力矩

刀軸螺母依據機台規格上緊到規定力矩。力矩不足導致刀片在切割負載下滑移，帶料寬度漂移；力矩過大可能造成刀軸彎曲（尤其是細軸徑）。

## 磨損識別與研磨時機

### 切割品質信號

| 症狀 | 可能原因 |
|------|---------|
| 毛邊高度超過材料厚度 10% | 刀刃磨損、間隙設定過小 |
| 切斷面出現撕裂紋路 | 間隙過大、刀刃崩缺 |
| 帶料邊緣出現波浪 | 壓入量不足、刀軸跳動過大 |
| 主軸電流持續升高 | 刀刃鈍化，需研磨 |

### 研磨時機原則

**不要等到完全失效才研磨。** 刀刃初期磨損時，每次研磨只需去除 0.05–0.1mm；等到嚴重磨損後才研磨，每次需去除 0.3–0.5mm 或更多，大幅縮短刀片可研磨次數。

**建議研磨觸發條件：**
- 毛邊高度超過材料厚度 8–10%
- 切割力（電流）較初期升高 15% 以上
- 肉眼可見刃口出現反光亮帶（接觸面積增大的標誌）

## 研磨作業規範

| 項目 | D2 / HSS 刀片 | 硬質合金刀片 |
|------|-------------|------------|
| 砂輪種類 | CBN 或陶瓷結合劑 Al₂O₃ | 金剛石砂輪（必須） |
| 冷卻液 | 充分供應，防止熱裂 | 高壓冷卻液，防止熱應力崩裂 |
| 研磨方向 | 上下刀片配對研磨 | 上下刀片配對研磨 |
| 研磨後公差 | ±0.002mm | ±0.002mm |
| 研磨後跳動 | TIR ≤ 0.005mm | TIR ≤ 0.003mm |

**重要：** 上下刀片必須成對研磨，研磨後重新配對使用，確保外徑差異在 ±0.002mm 以內。

**Q：D2 和 M2 有什麼實際差異？**
D2 泛用、性價比高，線速 < 150m/min 切碳鋼完全足夠。M2 熱穩定性更好，適合高速線或薄料。M42 含鈷，不鏽鋼高產量首選。

**Q：硬質合金值得投資嗎？**
切難切材料（矽鋼、AHSS、銅箔）且月產 > 10 萬米，幾乎必然更經濟。壽命 5–10 倍 + 換刀次數大減，月總成本低 15–25%。

**Q：分條刀可研磨幾次？**
視初始外徑與每次去除量而定。早期研磨（去除量 0.05–0.1mm）可最大化研磨次數，拖到嚴重磨損才磨，每次去除量大 3–5 倍，壽命大幅縮短。

**Q：硬質合金崩刃原因？**
幾乎都是安裝問題：刀軸跳動 > 0.003mm、間隔環精度不足、或間隙過小。不是材質問題。
